# 📊 Reportes del Proyecto

![Reports](https://img.shields.io/badge/Documentation-Reports-blue)
![Project](https://img.shields.io/badge/Type-Academic%20Project-lightgrey)

Esta carpeta contiene los **reportes de avance del proyecto** realizados durante el desarrollo del sistema.

Los reportes documentan el progreso del proyecto, incluyendo:

- análisis del problema
- diseño de la arquitectura
- decisiones técnicas
- implementación del sistema
- resultados de pruebas
- integración de nuevos patrones de diseño (Bridge y Decorator)

Estos documentos sirven como **registro del proceso de desarrollo del proyecto**.
