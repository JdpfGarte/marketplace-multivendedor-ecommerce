# 🎨 Frontend

![React](https://img.shields.io/badge/React-Frontend-blue?logo=react)

El frontend del proyecto corresponde a la **interfaz de usuario del sistema**, desarrollada con **React**.

Este módulo permite a los usuarios interactuar con la plataforma de manera visual e intuitiva.

---

# 🧩 Funcionalidades

El frontend permitirá:

- visualizar productos
- gestionar el carrito de compras
- realizar pedidos
- autenticarse en el sistema
- interactuar con el marketplace

---

# 🚀 Tecnologías utilizadas

- React
- JavaScript / TypeScript
- HTML / CSS
- Axios (consumo de API)

---

# 🔗 Integración con el backend

El frontend se comunica con la API mediante solicitudes HTTP:

- GET /productos
- POST /pedidos
- GET /usuarios
